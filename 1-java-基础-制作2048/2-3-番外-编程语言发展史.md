让我们看看编程语言的发展历史:

你只需要知道,随着时间的发展,总体而言与计算机交互的手段: 编程语言是越来越"简单"的,是越来越"面向用户友好的"即可.

编年史:

| 时间              | 编程语言                                              |
| ----------------- | ----------------------------------------------------- |
| 20 世纪 40 年代末 | 机器语言                                              |
| 20 世纪 50 年代初 | 汇编语言                                              
| 1972 年           | C 语言                                             
| 1983 年           | C++（最初名为 C with Classes，1983 年正式命名为 C++） |
| 1991 年           | Python                                             
| 1995 年           | Java                                               

基本上来说,Java和Python是同一时间发明的两套编程语言.从某种意义上来说,它们都比C++/C更"用户友好".
