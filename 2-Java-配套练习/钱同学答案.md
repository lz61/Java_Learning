### 三、循环练习（10 分钟）

1.**while 循环求和**

使用`while`循环计算 1 到 20 之间所有奇数的和并打印结果。(不要用if-else)

```
public class test {
    public static void main(String[] args) {
        int x=1;
        int result=0;
        while(x<=20){
            result=result+x;
            x=x+2;
        }
        System.out.println(result);
    }
}
```

2.**for 循环打印图案**

使用`for`循环打印 4 行由`*`组成的直角三角形，例如：

```plaintext
*
**
***
****
```

```
public class test {
    public static void main(String[] args) {
        for (int i = 1; i <= 4; i=i+1) {
            for (int j = 1; j <= i; j=j+1) {
                System.out.print("*");
            }
            System.out.println();
        }
    }
}
```
